Creating together

Responsive websites and applications for all purposes. Let's plan your new project together.

Please see below a few examples of applications and websites I have realized so far.

Famtravel

Famtravel is an application for families travelling in European cities. It is a crowdsourcing platform for parents who want to share information and look for it. The app, based on my idea, was originally created at LeWagon coding bootcamp with Ruby on Rails. We were a team of four students, I acted as the lead developer. Afterwards I continued working on the application and the first version was published in 2019.

Recipeper

I wanted a place to store and share recipes and created Recipeper for this purpose. In this Ruby on Rails application user can also save personal notes and links, modify the number of portions and send the recipe to his email with a simple click. A beta version was published in March 2022.

Superpower rental app

This Ruby on Rails MVC application was made at a LeWagon coding bootcamp during five days, together with three other students. It is practically a clone of the AirBnB site, we were just renting out superpowers rather than accomodation.

Kirjais Marina

A trilingual website made for a small company. I created a new WordPress theme according to the customer's wishes.The content can now easily be updated in WordPress.

JavaScript games

With the help of some great tutorials by Ania Kubow and Wes Bos I coded a few simple games with plain JavaScript. I added features, fixed some bugs and created the style for them.

Cocktail recipe app

Another excercise site that I coded and designed using Ruby on Rails. Search for a tasty drink or add your own favourite cocktail recipe.
